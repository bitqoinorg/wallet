# bitQoin Wallet

The world's first quantum-proof shield for your coins. Built on Solana and Ethereum using the Qonjoint protocol: both keys required, always.

## Features

- Qonjoint: both keys must sign every transaction, no exceptions
- Non-custodial: keys never leave your devices
- Solana: native 2-of-2 multisig
- Ethereum: Gnosis Safe 2-of-2 multisig
- 6-language support (EN, JA, ZH, DE, ES, AR)
- Dark and light mode
- Orb Markets explorer integration

## Stack

- React 18 + Vite + TypeScript
- Tailwind CSS
- @solana/web3.js (Solana)
- ethers.js + Gnosis Safe SDK (Ethereum)
- Public RPC (no API key required)
