# Changelog

## v0.1.0 (2025)

### Initial Release

- Qonjoint dual-key protocol: both keys required for every transaction
- Non-custodial vault generation with Ed25519 keypairs
- Solana SPL token support
- 6-language support: English, Japanese, Chinese, German, Spanish, Arabic
- Dark and light mode
- Orb Markets explorer integration for transaction history
- Landing page with 9 subpages (Why, How, Proof sections)
- FAQ page
- GitHub Actions CI/CD pipeline for GitHub Pages deployment
