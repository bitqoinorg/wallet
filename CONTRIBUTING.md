# Contributing to bitQoin Wallet

Thank you for your interest in contributing.

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/your-username/wallet`
3. Install dependencies: `pnpm install`
4. Start dev server: `pnpm dev`

## Rules

- No em-dashes anywhere in code, comments, or content
- Use "Qonjoint" not "multisig" or "2-of-2"
- Brand name is "bitQoin" (capital Q)
- Palette: only #FAFAF5, #F7931A, #1a1a1a
- All code and commit messages in English

## Commit Format

```
feat(wallet): short description
fix(wallet): short description
docs(wallet): short description
```

## Pull Requests

- Keep PRs focused on a single change
- Describe what the change does and why
- Test on both light and dark mode
- Test all 6 languages (EN, JA, ZH, DE, ES, AR)
